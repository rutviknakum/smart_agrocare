# 🌱 Smart Agrocare

An AI-powered mobile application for **crop disease detection and weather-based agricultural advisory**.

---

## 📱 Features

* 📸 Leaf Disease Detection using Deep Learning
* 🌦 Real-time Weather Integration
* 💊 Smart Advisory (Treatment & Prevention)
* 📊 User-friendly Mobile Interface

---

## 🧠 Tech Stack

* Flutter (Mobile App)
* Python (FastAPI / Flask)
* TensorFlow / PyTorch
* OpenWeather API

---

## 🏗 System Architecture

Mobile App → API → AI Model → Weather API → Advisory System

---

## 🚀 How It Works

1. Capture leaf image
2. Upload to backend
3. AI model detects disease
4. Weather data fetched
5. Advisory generated
